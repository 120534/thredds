Merge changes from branch A onto branch B in intelliJ

1. make sure you hav all changes locally : VCS / update
2. checkout branch B, if already checkout commit all changes
3. VCS / Git / Merge : Branches to Merge : select branch A
   If conflicts, they will be shown in dialog box : "Yours" = local branch = B, "Theirs" = changes from branch A
   (If no conflicts, will bring you to a commit dialog ??)
4. Commit your changes, now you are back to branch B with changes merged.
